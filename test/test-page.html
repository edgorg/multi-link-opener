Link Opener - Test Page

Use this page to test the extension. Select text in each block, right-click, and choose "Open links in selection".

Test 1: Raw URLs in text

Check out https://example.com and also visit https://google.com for more info. You might also like https://github.com which is great for developers.

Expected: 3 tabs open (example.com, google.com, github.com)

Test 2: Hyperlinked text (href differs from visible text)

Visit the free encyclopedia and the developer Q&A site for help.

Expected: 2 tabs open (wikipedia.org, stackoverflow.com)

Test 3: Mixed raw URLs and hyperlinks

Here's a raw link https://example.com and a hyperlinked one in the same selection.

Test 4: Duplicate links

Visit https://example.com here, and also https://example.com here again. Plus this link which is the same.

Expected (duplicates OFF): 3 tabs open

Settings Tests

Use the test blocks above with different settings:

Setting Test With Expected
Open mode: Current Test 1 Tabs open in current window, no group
Open mode: Group Test 1 Tabs open in current window, in a group labelled "Links"
Open mode: New window Test 1 New browser window opens with all tabs
Focus first tab: ON Test 1 Browser switches to first new tab
Focus first tab: OFF Test 1 Browser stays on current tab
